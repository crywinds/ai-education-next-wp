# 如何启动程序

## 方法一：使用 start.bat（推荐）

1. **直接双击** `start.bat` 文件
2. 程序会自动：
   - 检查 Node.js 是否安装
   - 检查 MySQL 服务是否运行
   - 检查端口 5200 是否被占用（如果被占用会自动关闭）
   - 检查并安装依赖包（如果需要）
   - 启动 Next.js 开发服务器

3. 启动成功后，在浏览器中打开：`http://localhost:5200`

## 方法二：使用 start-all.bat

1. **直接双击** `start-all.bat` 文件
2. 会显示一个菜单，选择：
   - **1** - 启动 AI 教育平台前端（Next.js）- 端口 5200
   - **2** - 启动所有服务
   - **3** - 退出

## 如果遇到乱码问题

如果打开 `start.bat` 或 `start-all.bat` 时看到乱码，请尝试以下方法：

### 方法 1：使用命令行手动启动

1. 按 `Win + R` 打开运行窗口
2. 输入 `cmd` 并按回车
3. 在命令行中切换到项目目录：
   ```batch
   cd /d F:\FIFA\new-next-wp
   ```
   （请根据您的实际路径修改）

4. 运行以下命令：
   ```batch
   npm run dev
   ```

### 方法 2：创建新的批处理文件

如果文件仍有问题，可以手动创建一个简单的启动文件：

1. 在项目根目录创建一个新的文本文件
2. 将文件重命名为 `run.bat`
3. 用记事本打开，输入以下内容：
   ```batch
   @echo off
   cd /d "%~dp0"
   npm run dev
   pause
   ```
4. 保存并双击运行

## 常见问题

### Q: 端口 5200 已被占用？
A: `start.bat` 会自动尝试关闭占用端口的进程。如果无法自动关闭，请：
- 打开任务管理器（Ctrl + Shift + Esc）
- 找到占用端口的 Node.js 进程并结束它
- 或者使用其他端口：修改 `package.json` 中的端口号

### Q: MySQL 服务未运行？
A: 程序会提示您启动 XAMPP Control Panel 并启动 MySQL 服务

### Q: 依赖包未安装？
A: 程序会自动运行 `npm install` 安装依赖包

### Q: Node.js 未安装？
A: 请先安装 Node.js：https://nodejs.org/

## 快速启动命令（命令行）

如果您习惯使用命令行，可以直接运行：

```batch
npm run dev
```

这会在端口 5200 启动开发服务器。
